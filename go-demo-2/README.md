# Go Demo 2 — Массивы, слайсы, циклы

## Темы
- Массивы (фиксированная длина) и слайсы (динамические)
- `len`, `cap`, `append`, `make`
- Распаковка слайса (`...`)
- Циклы `for` и `for range`
- Два стиля цикла: классический `for i := 0` и `for range`

## Итоговый проект
Счётчик транзакций — пользователь вводит транзакции одну за другой,
вводит `0` для завершения, программа считает итоговый баланс.
