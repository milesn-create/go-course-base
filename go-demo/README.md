# Go Demo — Основы языка

## Темы
- Структура программы, пакеты, `go build`, `go run`
- Переменные (`:=`, `var`, `const`), типы данных
- Ввод/вывод (`fmt.Scan`, `fmt.Printf`, `fmt.Scanln`)
- Условия (`switch`, `if/else`)
- Обработка ошибок (`errors.New`, `err != nil`)
- `panic()` — аварийное завершение программы
- `continue` в цикле

## Итоговый проект
Калькулятор ИМТ — пользователь вводит рост и вес,
программа считает индекс массы тела и выводит расшифровку.
Поддерживает повторный расчёт без перезапуска.
